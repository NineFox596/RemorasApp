# RemorasApp
# Pasos para la instalacion correcta del proyecto  
npx rn-new RemorasApp --nativewind --yarn  
cd RemorasApp  
yarn 
Tras creacion de proyecto, crear un archivo .env

# Importante
Dentro del archivo .env, colocar API_URL="tu_url".
No subir el archivo .env ni .gitignore.
Si en algun momento se planea usar mas de una url, pasaremos a usar el archivo .gitignore, y .env quedara obsoleto.
Este readme se ira expandiendo con las futuras dependencias que se vayan instalando.
